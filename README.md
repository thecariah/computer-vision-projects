# Computer Vision Projects

Este repositorio contiene proyectos desarrollados en el ramo de especialidad **Análisis de Video**, enfocado en **computer vision y deep learning** para analizar imágenes y videos.

## Proyectos

- **Tarea 1 – Image Retrieval**: Búsqueda de imágenes similares con ResNet18.
- **Tarea 2 – Video Summarization**: Generación de sumarios estáticos con K-Means y *baselines Random/Uniform*.
- **Proyecto final - Video People Counting**: Detección y conteo de personas en espacios públicos con YOLOv7.

## Stack

- PyTorch
- Torchvision 
- OpenCV 
- scikit-learn
- NumPy   
- Pandas
